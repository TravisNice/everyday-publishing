This theme is built for Everyday Publishing's own site.

== Description ==
This theme is built for speed and content delivery. It is lightweight, and
super fast to load. The aim of the theme is to increase the ad quality on
Google AdWords resulting in cheaper CPC.

== Installation ==
This theme is only available through the Everyday Publishing GitHub Repository.
